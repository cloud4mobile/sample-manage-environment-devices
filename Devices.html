How-To: Query the Environment's Devices

This example shows how to query the devices of an environment. To proceed, the following information is required and will be used to create an authorization token which grants access to the desired information through our Web API.

  • Consumer Key is the user's unique key. This key gives the user access to a specific environment. .
  • Consumer Secret is the user's secret. All requests related to the environment (identified by the user's consumer key) are signed using this secret..
